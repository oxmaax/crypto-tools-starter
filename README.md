# crypto-tools-starter

a small collection of simple crypto utilities built while learning python and automation.

## tools included

- **bitcoin price checker**
  - fetches the current btc price in usd from the coingecko api.
- **basic apy calculator**
  - quickly estimates future balance based on a starting amount and annual yield.

## requirements

- python 3.9+
- `requests` library

install dependencies:

```bash
pip install -r requirements.txt
